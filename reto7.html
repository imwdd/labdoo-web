DOCUMENTACIÓ WEB DEL PROJECTE

Els objectius d'aquest repte són:

  • El lloc web haurà de tenir diverses pàgines.
  • El pressupost que vàreu implementar al "Repte 2 - Disseny i implementació de la infraestructura física i lògica", caldrà implementar-ho en una taula HTML, no com a imatge.
  • Cada pàgina del lloc web inclourà el menú i es podrà accedir a totes les pàgines.
  • Tota la documentació del projecte s'ha d'incloure al lloc web.
  • La web ha d'incloure com a mínim un vídeo. Ha de ser fàcil lectura i visual, inclougueu elements gràfics.
  • L'estil de totes les pàgines ha de ser uniforme, a través d'un únic fitxer CSS per a tot el lloc web. Recordeu que podeu personalitzar estils d'elements concrets o de grups d'elements fent ús de les ID i les CLASS.
  • S'ha d'incloure un formulari on els usuaris puguin introduir peticions d'infraestructures. El formulari no ha de desar les dades, només s'ha d'implementar la pàgina web i ha de contenir, com a mínim, els següents camps:
  • Dades personals de l'usuari (nom, cognoms).
  • Dades de contacte (telèfon, correu electrònic).
  • Número d'ordinadors que el client vol incloure.
  • Si la connexió serà per cable, wi-fi o ambdues.
  • Número de sales que es faran servir a la instal·lació (un màxim de 10 sales). La selecció s'ha de fer amb un camp desplegable.
  • Text lliure de comentaris addicionals.
  • Cal seleccionar el tipus de camp més adient per a recollir cadascuna de les dades anteriors i aplicar les restriccions indicades. Tots els camps indicats a excepció de l'últim han de ser obligatoris.

Tecnologies

Visual Studio Code

Visual Studio Code (VS Code) és una eina de desenvolupament versàtil i eficient creada per Microsoft. VS Code es destaca per la seva lleugeresa i adaptabilitat, oferint un entorn d'escriptura de codi amb funcionalitats avançades. Aquest editor de text proporciona ressaltat de sintaxi, suggeriments de codi i integració amb múltiples llenguatges, facilitant als desenvolupadors escriure i depurar codi de manera efectiva.

HTML5

HTML, que significa "Llenguatge de Marcatge d'Hipertext" en anglès (Hypertext Markup Language), és un estàndard utilitzat per crear i dissenyar pàgines web. Va ser desenvolupat pel World Wide Web Consortium (W3C) i s'utilitza per estructurar el contingut d'una pàgina web mitjançant l'ús d'elements i etiquetes. En essència, HTML proporciona un conjunt d'etiquetes que descriuen l'estructura lògica d'una pàgina web. Aquestes etiquetes delimiten diferents parts del contingut, com ara capçaleres, paràgrafs, llistes, enllaços, imatges, formularis, entre d'altres.

CSS3

CSS, que significa "Full de Estils en Cascada" en anglès (Cascading Style Sheets), és un llenguatge de marcatge utilitzat juntament amb HTML per estilitzar i donar format a les pàgines web. La funció principal de CSS és separar la presentació del contingut d'una pàgina web, permetent als desenvolupadors controlar l'aparença i la disposició dels elements HTML de manera més eficient.

JavaScript

JavaScript és un llenguatge de programació àmpliament utilitzat en el desenvolupament web per afegir interactivitat i dinamisme a les pàgines. Va ser creat inicialment per Netscape i s'ha convertit en un estàndard per als navegadors web, permetent als desenvolupadors crear aplicacions i experiències interactives a les pàgines web. JavaScript és un llenguatge de programació del costat del client, el que significa que s'executa al navegador de l'usuari final enlloc de fer-ho al servidor.

  • Nosaltres per la web l'hem utilitzat per crear un script que ens permet copiar el text que emmagatzem a una determinada etiqueta p.
GitHub

GitHub és un servei basat en el núvol que allotja un sistema de control de versions (VCS) anomenat Git. Aquest permet als desenvolupadors col·laborar i fer canvis en projectes compartits, alhora que mantenen un seguiment detallat del seu progrés.

  • * El control de versions és un sistema que ajuda a rastrejar i gestionar els canvis realitzats en un fitxer o conjunt de fitxers.

GitHub Desktop

GitHub Desktop és una aplicació gratuïta i de codi obert que us ajuda a treballar amb fitxers allotjats a GitHub o altres serveis d'allotjament de Git. GitHub Desktop és útil per a usuaris principiants i avançats. Els beneficis inclouen:

  • Comença fàcilment
  • Si ets nou a Git i GitHub, pots trobar més fàcil utilitzar GitHub Desktop que utilitzar Git a la línia de comandes. GitHub Desktop té una interfície gràfica d'usuari que simplifica els comandaments i t'ajuda a visualitzar els canvis.
  • Troba comandaments
  • GitHub Desktop té una interfície visual, pots accedir fàcilment a comandaments menys comuns de Git, com ara triar quines línies canviades incloure en un commit o afegir un co-autor a un commit, sense necessitat de memoritzar o buscar la sintaxi.
  • Segueix les millors pràctiques
  • GitHub Desktop fomenta les millors pràctiques. Per exemple, t'ajuda a crear un històric de commits precís i fàcil de seguir perquè altres col·laboradors en un projecte puguin revisar fàcilment la teva feina.
  • Integra't amb GitHub
  • GitHub Desktop està dissenyat específicament per utilitzar-se amb GitHub. Per exemple, pots autenticar-te ràpidament a GitHub.com o a GitHub Enterprise Server, sense necessitat d'utilitzar un gestor de credencials separat. També pots fer el check-out d'una sol·licitud de pull per executar comprovacions sense necessitat d'obrir el teu navegador.

PALETA DE COLORS

Per escollir la paleta de colors ens vam recolzar a l'eina de coolors.co. Coolors és una aplicació web que facilita la generació de paletes de colors de manera intuïtiva i eficient. Amb aquesta eina, dissenyadors, desenvolupadors i altres professionals poden descobrir i experimentar amb combinacions de colors harmonioses per als seus projectes. Coolors ofereix la funcionalitat de generar paletes automàticament, proporcionant combinacions equilibrades amb tan sols prémer un botó. A més, els usuaris poden explorar paletes creades per la comunitat, oferint una font d'inspiració i possibilitant la trobada de combinacions de colors populars i atractives.
