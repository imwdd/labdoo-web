INSTAL·LACIÓ DELS SERVIDORS I CLIENTS

Respecte a la implementació real del projecte, la part que farem en sistema operatiu real (virtualbox) serà la part de servidors i un client Windows:

  • Hem de definir i justificar quines característiques tindrà el servidor i justificar quin sistema operatiu tindrà.
  • Hem de definir i justificar quines característiques haurien de tenir els clients i justificar quin sistema operatiu tindrà.
  • Haurem de configurar un domini per aquest servidor.
  • Haurem de fer un document amb el procés d’instal·lació del servidor i del client

Llista de limitacions:

  • S’ha de respectar les característiques d’instal.lació mínimes que recomanin els diferents servidors.
  • El nom de la màquina es dirà SERVER-NºGRUP (per exemple SERVER-1)
  • Si instal·leu més d’un server anomeneu-los SERVER-1-1, SERVER-1-2 etc..
  • El nom del client serà CLIENT-NºGRUP (per exemple CLIENT-1)

COMPARATIVA

Entenent les necessitats que té la vostra empresa us oferim tres sistemes operatius en xarxa.

El primer és Windows Server 2016, destacat per la fàcil configuració gràcies a la seva interfície gràfica, és compatible amb la majoria de programes que s'utilitzen actualment. El seu desavantatge són els preus que pot arribar a tenir, és un sistema operatiu en xarxa privat, per la qual cosa has de comprar una llicència.

El segon és Ubuntu Server 22.04 LTS, es destaca per ser un sistema operatiu en xarxa de codi obert, cosa que significa que no cal comprar una llicència i de cara a la personalització, tens més opcions per adaptar el sistema a la teva manera. La seguretat és elevada a diferència de Windows Server i sol tenir més rendiment que Windows Server.

El tercer és UNIX es destaca per ser un sistema operatiu en xarxa que permet diversos usuaris utilitzar-lo alhora. És de codi obert i sol tenir una alta estabilitat i seguretat.

WINDOWS SERVER

Pel projecte utilitzarem el sistema operatiu Windows Server 2016 degut a l’equilibri entre la facilitat que ofereix la interfície gràfica i la seguretat que ens proporciona.

Procesador Memoria RAM Emmagatzament Adaptador de xarxa
Processador de 64 bits a 1,4 GHz (Compatible amb el conjunt d´instruccions x64) 512 MB (2 GB per a l'opció d'instal·lació Servidor amb Experiència d'escriptori) 32GB Un adaptador Ethernet amb un rendiment d'almenys 1 gigabit per segon

UBUNTU SERVER

Ubuntu Server 22.04 LTS l'utilitzarem de backup per si el servidor principal, el Windows Server falla. De moment, no hi farem cap configuració.

Requisits mínims
Procesador Memoria RAM Emmagatzament Port USB
Processador de doble nucli a 2 Ghz o superior. Com a mínim 4 GB de ram. 25 GB Puerto USB i una memória USB de al menys 4 GB de capacitat.
