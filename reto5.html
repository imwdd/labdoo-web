SANEJAMENT DE PORTÀTILS LABDOO

ETAPES DEL SANEJAMENT DE PORTÀTILS

El sanejament de portàtils a Labdoo es realitza seguint les següents etapes:

  • 1. Etiquetat
  • Etiquetar el portàtil dins de la plataforma Labdoo: Posar el portàtil a estat S1
  • 2. Neteja
  • Netejar i preparar físicament el portàtil: Neteja del teclat, pantalla, ventilador,... Fent servir brotxes, drap i/o aire comprimit
  • 3. Sanejament
  • Prepararem a nivel de software del portàtil: ho expliquem en detalla la secció següent.
  • 4. Comprovació
  • Comprovar que tot funciona correctament al portàtil: farem comprovació a nivell de teclat, video, càmara, les aplicacions obren correctament i es pot treballar amb elles.

SANEJAMENT A NIVELL DE SOFTWARE

Existeixen 2 tipus de sanejament i "puesta a punto" a nivel de software del portàtil Labdoo:

  • A) El sanejament pas a pas:
  • - Bootegem un USB amb GParted i esborrem totes les particions.
  • 1. Prepara un USB booteable con Labtix
  • 2. Engega el portàtil des del USB i obre gparted.
  • 3. Esborra totes les particions i aplica els canvis.
  • Si et queden dubtes aquí tens instruccions més precises de com fer-ho: https://platform.labdoo.org/content/creation-labtix-bootable-usb
  • - Fem la neteja segura de dades del disc dur fent servir la següent comanda per sobreescriure el disc 2 vegades:

sudo shred /dev/sda -f -v --iterations=2

  • - Instal·lem una distribució de Linux adient als hardware que tenim a la màquina: Lubuntu, Linux Mint, Puppy Linux...
  • B) Sanejament fent servir Labtix:

Labtix és una distro de Linux creada per Labdoo, basada en la distro Antix que conté una serie de programes i eines útils per chequejar l'estat del portàtil i fer sanejament a nivell de software. Els pasos a seguir són:

  • - Bootegem un USB amb Labtix
  • - Arrancar el portàtil des del USB
  • - Obre el programa StressCPU: comprova la temperatura
  • - Obre el programa Disk Check i fes el test ràpid
  • - Obre el programa zzzFM:
  • Selecciona el disc extern on tinguis les imatges de Labdoo al quadre de la dreta.
  • Selecciona la carpeta on tens la imatge de clonezilla que vols clonar al disc dur intern. Clica botó dret de ratolí i selecciona "Labtix Install"
